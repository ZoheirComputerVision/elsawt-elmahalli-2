# دستور المشروع — الصوت المحلي

## 1. الرؤية
منصة إعلامية جهوية ذكية تغطي ولاية تيارت، ولاية تيسمسيلت، ودائرة قصر الشلالة، تقدم محتوى إخباري موثوق، دليل اقتصادي متكامل، وأدوات تفاعلية للمواطن والمسؤول.

## 2. المبادئ التوجيهية
- **الموثوقية**: جميع الأخبار تخضع للتدقيق قبل النشر
- **الجهوية**: التركيز على الشأن المحلي لثلاث مناطق التغطية
- **الشفافية**: سجل تدقيق كامل لكل عملية تحرير ونشر
- **التطوير المستمر**: إضافة الميزات بشكل تدريجي ومنهجي

## 4. اتفاقيات التسمية
- **المجلدات**: `kebab-case` (مثال: `economic-directory`)
- **المكونات**: `PascalCase` (مثال: `NewsService`)
- **المسارات**: `kebab-case` (مثال: `/api/economic-directory`)
- **قواعد البيانات**: `snake_case` (مثال: `featured_stories`)

## 5. مبادئ الحدود (Boundaries)
- لا يتم تعديل الصفحة الرئيسية إلا ضمن خطة معتمدة
- لا يتم تعديل سير العمل التحريري بعد الاستقرار
- لا يتم تعديل محرك البحث الأساسي بعد الاستقرار
- كل ميزة جديدة تبدأ بوثيقة تصميم في `/features/`

## 6. مسار اتخاذ القرارات
```
اقتراح → وثيقة في /features/
  → مراجعة → موافقة → تنفيذ
  → اختبار → نشر → توثيق
```
